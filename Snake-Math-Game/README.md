## 🎮 Introduction
**"Rắn Săn Mồi - Luyện Phép Nhân"** (Snake Math Trainer) is an interactive, browser-based educational game designed to help children (specifically 3rd to 5th graders) practice their multiplication tables playfully. By combining the classic gameplay of Snake with rapid mental math challenges, players improve both their reaction time and foundational math skills simultaneously.

## ✨ Features
- **Dynamic Question Generation:** Questions encompass multiplication tables from 2x1 to 9x9.
- **Smart "Food" Placement:** Instead of standard "apples," 4 distinct numbered foods appear on the board. Only one is the correct answer; the rest are purposefully crafted "trick" answers to challenge the user's focus.
- **Kid-Friendly Aesthetics:** The UI employs a soothing color palette, soft rounded corners, bubbly fonts (Fredoka One), and subtle particle animations to keep children engaged without over-stimulation.
- **Responsive Controls:** Arrow keys (or WASD) navigate the snake, buffered by a custom input queue to prevent rapid illegal U-turns and frustration.
- **Progressive Difficulty:** Each correct answer extends the snake's tail, increasing the spatial constraint challenge while the speed remains manageable for younger audiences.

## 🛠️ Technical Details
- **Rendering Context:** Entirely rendered using the standard HTML5 `<canvas>` API with a secondary grid overlay.
- **Game Loop Engine:** Operates on standard `setInterval()` frame ticks manually tuned for children's reaction speeds.
- **No Dependencies:** Strictly written in Vanilla JavaScript, HTML, and CSS (Zero libraries used).
